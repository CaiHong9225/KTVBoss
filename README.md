# K歌打Boss

一周黑客松作品方案，面向 Android TV / K 歌机顶盒场景。

## 项目定位

`K歌打Boss` 是一个把唱歌过程实时映射成战斗反馈的互动作品。用户不是在单纯唱歌，而是在用自己的声音输出伤害、积攒能量、触发技能，最终击败屏幕中的 Boss。

这个项目适合用来参加「一人一周 AI 编程」类比赛，因为它具备以下特点：

- 第一眼就能理解，传播成本低
- 视觉和交互冲击力强，容易出视频效果
- 与 K 歌机顶盒业务高度贴合，有现实延展性
- 可以只做一个可运行的核心 demo，不需要大而全
