# K歌打Boss 页面草图

## 1. 首页

页面目标：让用户 5 秒内理解玩法并开始挑战。

布局建议：

- 顶部：作品标题 `K歌打Boss`
- 中部左侧：一句话说明
- 中部右侧：Boss 海报或动态立绘
- 底部主按钮：`开始挑战`
- 底部次按钮：`玩法说明`

大屏原则：

- 焦点按钮不超过 2 个
- 主按钮尺寸足够大
- 首页不要放太多文字

## 2. Boss 登场页

页面目标：建立情绪和戏剧性。

布局建议：

- 居中显示 Boss 名称
- 下方显示一段 AI 生成的挑衅台词
- 2 秒后自动进入战斗，也可遥控确认跳过

示例台词：

`我是包厢噪音吞噬者。今天，你的副歌会变成我的能量。`

## 3. 战斗页

这是核心页面，优先保证阅读性和反馈节奏。

```text
┌────────────────────────────────────────────┐
│ Boss 名称                      Boss HP     │
│ [Boss 立绘 / 动效区域]                     │
│                                            │
│                                            │
│        攻击特效 / 受击数字 / 技能动画      │
│                                            │
│ 歌词区 / 当前段落提示                      │
│                                            │
│ Player HP   Energy   Combo   状态提示      │
└────────────────────────────────────────────┘
```

区域说明：

- 上方：Boss 名称、血条、状态
- 中间：Boss 立绘和战斗特效
- 中下：歌词区或节奏提示区
- 下方：玩家状态栏，包含能量、combo、攻击反馈

## 4. 结算页

页面目标：让用户觉得「这一局值了」。

布局建议：

- 顶部：胜利 / 失败标题
- 中部左侧：本局数据摘要
- 中部右侧：AI 点评和玩家称号
- 底部按钮：`再来一次`、`返回首页`

推荐展示字段：

- 玩家称号
- 最大连击
- 高能时刻
- Boss 最后一句话
- 本局一句话总结

## 5. 视觉风格建议

主题风格可选其一：

- 赛博包厢风：霓虹、紫蓝、电子粒子
- 热血番战斗风：红黑、速度线、爆炸字效
- 音乐舞台风：灯光扫射、频谱、聚光灯

推荐优先选 `赛博包厢风`，因为最适合 K 歌与战斗融合。

## 6. 必做动态反馈

为了让作品看起来真的酷，以下反馈至少要具备：

- 发声时玩家区域发光
- 攻击命中时 Boss 闪白或震动
- 连击增长时数字放大
- 必杀技时全屏遮罩和镜头切换
- 结算页称号出现时有仪式感动画

## 7. 远距离可读性要求

- 关键数值字号大于普通说明文字
- 血条和能量条颜色对比明显
- 画面最多只保留一个视觉主焦点
- 状态提示尽量使用短词，如 `暴击`、`连击`、`反击`
